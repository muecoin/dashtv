Help

This simple monitoring tool calculates the estimated income [excl. hosting] of your MUE Masternodes and displays it in a clear way. The data gets refreshed every minute automatically. To learn more about the MUE project visit MonetaryUnit.org.

Url parameters.
Please note, the custom parameters below are currently not supported.

Default parameters:
500000 MUE (1MN) and USD as fiat currency.

Custom parameters:
curr=EUR or CNY,GBP,CAD,RUB,HKD,JPY,AUD,CHF,PLN
addr=address1,address2,address3... (too many addresses will take some time)
OR
value=customMueAmount

Examples:

https://muetv.com/muetv/#curr=EUR&addr=7puXyuD4mt1vudZzMya3c7wfzym9S4ktXP,7gdKrsRzShBS4NSUSrk2VhMcPDKF2MvLWK https://muetv.com/muetv/#curr=CAD&value=2513
